# Krzen -- Image Compressor, Resizer & Format Converter

**[-> Use Krzen (live tool)](https://krzen.com/)**

Krzen is a free client-side image compressor that processes images entirely in your browser using the Canvas API. Compress JPEG, PNG, and WebP images with adjustable quality, resize to custom dimensions, convert between formats including AVIF, and compare before/after with a visual slider. Your images never leave your device.

## Features

- Adjustable quality slider from 1-100 with real-time preview
- Output format selection: JPEG, WebP, PNG, AVIF
- Custom width and height resizing with aspect ratio preservation
- Visual before/after comparison with draggable slider
- Bulk compression for processing multiple files simultaneously
- Page load impact calculator showing time savings across connection types
- 100% client-side -- your data never leaves your browser
- Open source -- inspect the code yourself

## Tech Stack

- Vanilla JavaScript (no frameworks, no build step)
- Static HTML hosted on GitHub Pages
- Cloudflare DNS + SSL
- Zero dependencies, zero tracking, zero cookies
