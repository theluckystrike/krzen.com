# Image Compressor — Compress, Resize & Convert Images in Your Browser

**[Compress Images →](https://krzen.com)** | [About](https://krzen.com/about.html) | [Blog](https://krzen.com/blog/)

Image Compressor lets you compress, resize, and convert images entirely in your browser. Drag and drop any JPEG, PNG, or WebP file, adjust the quality slider, and see a real-time before/after comparison with file size reduction stats. Convert between formats (JPEG to WebP, PNG to JPEG) and download optimized images instantly. Your images never leave your device.

## Features

- **Drag-and-drop upload** — drop images or click to browse, supports JPEG, PNG, and WebP
- **Quality slider** — fine-tune compression level with real-time preview
- **Format conversion** — convert between JPEG, PNG, and WebP with one click
- **Before/after comparison** — side-by-side view with file size reduction percentage
- **Batch processing** — compress multiple images at once and download as a ZIP

## How It Works

Drag one or more images onto the drop zone (or click to select files). The compressor reads your images using the Canvas API and applies client-side compression at your chosen quality level. The before/after panel shows the original and compressed versions side by side with file size stats. Use the format dropdown to convert between JPEG, PNG, and WebP. When satisfied, download individual files or all at once as a ZIP. Everything runs in your browser — images never touch a server.
