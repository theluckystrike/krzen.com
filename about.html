About krzen

krzen.com is a free image compression tool that runs entirely in your browser using the Canvas API. No images are uploaded to any server. Everything is processed locally on your device.

Features

  • Drag and drop or file picker for JPEG, PNG, and WebP
  • Quality slider (1-100) for precise compression control
  • Output format selection: JPEG, WebP, PNG
  • Resize with aspect ratio preservation
  • Before/after slider comparison
  • Compression stats: original size, compressed size, savings
  • Bulk mode: drop multiple images, compress individually

How It Works

The tool uses the HTML5 Canvas API to read your image, optionally resize it, then re-encode it using canvas.toBlob() with your chosen format and quality settings. This is the same compression pipeline used by modern browsers for native image rendering.
